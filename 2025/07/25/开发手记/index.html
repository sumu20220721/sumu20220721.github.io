开发手记

发表于2025-07-25|更新于2025-07-27
|总字数:1.3k|阅读时长:3分钟|浏览量:

250725

突发奇想能不能搭一个自己的博客。

其实也不是第一次有这个想法了，以前也想过好几次，只是在网上大致搜了搜，又要买域名又要买服务器，花钱且麻烦。

最近上课摸鱼，实在想干点什么，于是乎这个想法又浮了上来。

按以前大概就在拖延拖延中不了了之了。

但这几天消磨了太多时间在无意义的事情上，想做出改变。

硬核狠人的共同特征是超强执行力。

所以再次想到这件事，马上爬起来找资料。

其实本人对编程什么的一窍不通。

最开始找的是一个用Hexo+Gitee搭的教程。

做到一半看到评论区说Gitee已经不能用了。

只好重新找，换Github。

自以为前面做好了，只要把搭仓库的部分换成Github。就跳过了前面的部分。

做来做去还是不行，原来前面也有一些不一样的地方。

反复做反复倒查，弄了半天，因为两个教程不一样，有些前面的步骤其实落下了很多。

还不如完全重新做。

配置ssh key这步，这里不要加速。

做到了本地部署，但上线不了。

hexo d之后只显示validating config，后面都没有，看评论区也解决不了。

明天继续。

bash的ctrl+c是结束进程！！千万不要顺手以为是copy！！！！！

250726

重新下了nodejs。

用Deepseek帮忙看哪里出错了，很好用。

千奇百怪的错误，不知道类似的埋藏在哪里的回帖或评论或弹幕。

hexo d还是不行。

倒腾两个小时，最后发现了，用记事本编辑_config.yml之后我直接关闭了窗口，但是这其实并没有关闭文件，根本没保存……

《你有没有被sb的BUG困扰过很久》

遇到了新的问题，加速的话显示ssh错误，不加速的话显示连不上Github。

把HTTP协议换成SSH协议也不对。

还是听Deepseek的，配置Git使用系统证书存储（推荐）

在较新版本的Git中，可以设置Git使用操作系统的证书存储：

git config –global http.sslBackend schannel

（Windows系统）

虽然原理一点不知道，但是真的解决了耶！

然后电脑就没电了（草）

中午充上电，终于成功上线了！下午搞美化w

美化的第一个问题是：用anzhiyu还是butterfly。

纠结了一会儿选了anzhiyu，指南看得不是很明白，捣鼓了半天收效甚微。

而且anzhiyu的教程视频好少。也不知道是不是自己没找到。

看不到自己想要的效果，正反馈不足。

还是用butterfly吧。

找到了很全的文字教程和视频。

下午成功安装了butterfly主题。

晚上想做一下目录栏、头像和背景图。

按照教程做，慢慢看懂了一点点。

也悟了一点当时看anzhiyu没看明白的地方。

_config.butterfly.yml改了，还学了相对路径的写法。可页面没有相应的变化。

Deepseek：没有正确启用配置覆盖

检查Hexo根目录的_config.yml中是否指定了Butterfly的配置文件：

theme: butterfly

butterfly: _config.butterfly.yml # 这一行很重要，告诉Hexo使用这个文件作为主题的配置覆盖

做出目录栏和头像。

修改背景图为海上钢琴师。

添加打字滚动副标题。

250727

上午完成了一些基础设置。

包括文章页基础配置，尤其是分页按钮；页脚设置；简繁转换；侧边栏部分设置。删掉了不想要的一些部分。

想起做个人博客的引子之一，22年写文查东西的时候发现了一个叫“昨日当年”的网站，好像是谁的个人博客，零零碎碎地记录着自己的东西。

现在再找，那个网站已经没有了。

中午继续。

搜索栏已解决：把主题配置文件中 search 的 use 配置为 local_search。

网站背景图已解决：可能是选择的图片网站有防盗设置，所以不能引用。

标准写法应为background: “url(https://…)”，实操发现不需要，可能是浏览器容错，直接写图片 URL 会被自动转换为 url(…) 格式。

下午。

文章头图设置成功。

侧边栏优化：保留个人卡片，公告栏换行使用《br》，归档，分类。

首页卡片布局为6：瀑布流布局—封面在上，信息在下。

禁止转载：在share的use：留空。

添加文章编辑按钮：开启post_edit并填写url: github.com/user-name/repo-name/edit/branch-name/subdirectory-name/。

添加页脚文字：住住的未满阁楼。

准备做的魔改：一图流；分类磁贴和文章置顶轮播图；自定义字体；导航栏；电子钟等。

一图流制作完毕，代码没遇到什么困难，主要的困难是选图。

今天就先到这里。
